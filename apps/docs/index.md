## Quick Start

```bash
# Clone the repository
git clone https://github.com/DaScient/Think-and-Grow-Rich.git
cd Think-and-Grow-Rich

# Run the interactive setup script
./scripts/setup.sh

# Or manually bootstrap:
pnpm install
pnpm dev
```
